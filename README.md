# Live chat
Simplest datepicker
## Pre-request
Should be installed:
-  [Node js](https://nodejs.org/en/)
- [Git](https://git-scm.com/downloads)
## To start
```
  git clone https://github.com/Julyme3/datepicker.git
```
```
  yarn install
  ```
```
  yarn serve
```

## Stack:
- Native JS
- LESS
- momentjs
